# Complete Guide: Installing Python on Windows, macOS, and Linux

This guide provides a clear, step-by-step explanation of how to install
**Python** on all major operating systems.

------------------------------------------------------------------------

## 1. Installing Python on Windows

### Step 1: Download Python

1.  Visit the official Python website: https://www.python.org
2.  Go to the **Downloads** section and download the recommended version
    for Windows.

### Step 2: Run the Installer

1.  Open the downloaded installer file.
2.  **Important:** Check the option **Add Python to PATH**.
3.  Click **Install Now** and wait for the installation to finish.

### Step 3: Verify Installation

Open CMD or PowerShell and run:

    python --version

If a Python version number appears, the installation was successful.

------------------------------------------------------------------------

## 2. Installing Python on macOS

### Method 1: Using the Official Installer (Recommended)

1.  Go to Python.org.
2.  Download the macOS installer from the **Downloads** page.
3.  Run the `.pkg` file and follow the installation instructions.

### Method 2: Using Homebrew (For advanced users)

If Homebrew is not installed, install it first:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Then install Python:

    brew install python

### Verify Installation

    python3 --version

------------------------------------------------------------------------

## 3. Installing Python on Linux

Most Linux distributions already include Python by default.\
To check:

    python3 --version

### Ubuntu / Debian

    sudo apt update
    sudo apt install python3 python3-pip

### Fedora

    sudo dnf install python3 python3-pip

### Arch Linux

    sudo pacman -S python python-pip

------------------------------------------------------------------------

## 4. Installing pip (If needed)

    python3 -m ensurepip --upgrade

------------------------------------------------------------------------

## 5. Creating a Virtual Environment (venv)

### Create the environment

    python3 -m venv myenv

### Activate the environment

**Windows:**

    myenv\Scripts\activate

**macOS / Linux:**

    source myenv/bin/activate

------------------------------------------------------------------------

## 6. Installing a Sample Python Package

    pip install requests

------------------------------------------------------------------------

## Conclusion

You have successfully installed Python on your system.\
You can now start writing scripts, installing libraries, and building
applications.
