# üêç Setting Up Visual Studio Code(Windows, Mac OS, Linux)

Works on **Windows**, **macOS**, and **Linux**

This guide explains how to install and configure Visual Studio Code (VS
Code) for Python development across all major operating systems.

------------------------------------------------------------------------

## üì• 1. Install Python

Before setting up VS Code, make sure Python is installed.

### Windows / macOS / Linux

Download Python from the official website:\
https://www.python.org/downloads/

Verify installation:

    python --version

or

    python3 --version

------------------------------------------------------------------------

## üíª 2. Install Visual Studio Code (VS Code)

Download VS Code (all platforms):\
https://code.visualstudio.com/

### Windows

During installation, enable: - Add to PATH\
- Open with Code (optional)

### macOS

Drag VS Code into the Applications folder.

### Linux

Depending on your distro:

#### Ubuntu / Debian

    sudo apt update
    sudo apt install code

#### Fedora

    sudo dnf install code

#### Arch Linux

    sudo pacman -S code

------------------------------------------------------------------------

## üß© 3. Install Required Extensions

Open VS Code ‚Üí Click **Extensions** ‚Üí Install:

-   Python (Microsoft)\
-   Pylance\
-   Jupyter\
-   Code Runner (optional)

------------------------------------------------------------------------

## ‚öôÔ∏è 4. Select Python Interpreter

1.  Press `Ctrl + Shift + P`\

------------------------------------------------------------------------

## üåê 5. Setting Up a Virtual Environment

### Windows

    python -m venv venv
    venv\Scripts\activate

### macOS / Linux

    python3 -m venv venv
    source venv/bin/activate

Select interpreter again in VS Code after activation.
