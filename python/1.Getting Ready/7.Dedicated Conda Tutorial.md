# Complete and Dedicated Conda Tutorial (Linux, Windows, macOS)

## Introduction
Conda is a powerful package and environment management system used widely in data science, machine learning, Python development, and scientific computing. It allows you to create isolated environments, install packages with specific versions, and manage dependencies cleanly.

---

## 1. Installing Conda

### 1.1 Installation Options
- **Anaconda**: Full distribution with 1500+ scientific packages (large size).
- **Miniconda**: Lightweight installer with Conda only (recommended).

### 1.2 Installation by OS

#### **Linux**
```bash
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
source ~/.bashrc
```

#### **macOS**
```bash
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-x86_64.sh
bash Miniconda3-latest-MacOSX-x86_64.sh
source ~/.zshrc
```

#### **Windows**
1. Download Miniconda or Anaconda installer from the official website.
2. Complete installation via GUI.
3. (Optional but recommended) Check: *"Add Miniconda to PATH"*.

## 2. Core Concepts

### 2.1 Environments
Environments allow multiple isolated sets of Python versions and packages.

### 2.2 Channels
Channels are package sources.
Common channels:
- `conda-forge`
- `defaults`

---

## 3. Environment Management

### 3.1 Create a New Environment
```bash
conda create -n myenv python=3.11
```

### 3.2 Activate Environment
```bash
conda activate myenv
```

### 3.3 Deactivate Environment
```bash
conda deactivate
```

### 3.4 Remove Environment
```bash
conda remove -n myenv --all
```

### 3.5 List All Environments
```bash
conda env list
```

---

## 4. Package Management

### 4.1 Install Packages
```bash
conda install numpy
```
From conda-forge:
```bash
conda install -c conda-forge numpy
```

### 4.2 Remove Packages
```bash
conda remove numpy
```

### 4.3 Update Packages
```bash
conda update numpy
```

### 4.4 Search for Packages
```bash
conda search numpy
```

---

## 5. Working With YAML Environment Files

### 5.1 Export Environment
```bash
conda env export > environment.yml
```

### 5.2 Create Environment From YAML
```bash
conda env create -f environment.yml
```

### 5.3 Update Environment From YAML
```bash
conda env update -f environment.yml
```

---

## 6. Using Conda Environments with Jupyter Notebook

### 6.1 Install ipykernel
```bash
conda install ipykernel
python -m ipykernel install --user --name myenv --display-name "My Conda Env"
```

### 6.2 Remove a Kernel
```bash
jupyter kernelspec remove myenv
```

---

## 7. Advanced Conda Tips

### 7.1 Clean Cache
```bash
conda clean --all
```

### 7.2 Manage Channels
Add a channel:
```bash
conda config --add channels conda-forge
```

List channels:
```bash
conda config --show channels
```

### 7.3 Channel Priority
```bash
conda config --set channel_priority strict
```

---

## 8. Common Issues & Solutions

### Issue: Slow package installation
**Solution:** Give `conda-forge` highest priority.

### Issue: Cannot activate environment (Linux/macOS)
**Solution:**
```bash
source ~/miniconda3/etc/profile.d/conda.sh
```

### Issue: UnsatisfiableError
**Solution:**
- Check version conflicts.
- Try switching channels.
