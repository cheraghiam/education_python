# 🐍 Setting Up Visual Studio Code(Windows, Mac OS, Linux)

Works on **Windows**, **macOS**, and **Linux**

This guide explains how to install and configure Visual Studio Code (VS
Code) for Python development across all major operating systems.

------------------------------------------------------------------------

## 📥 1. Install Python

Before setting up VS Code, make sure Python is installed.

### Windows / macOS / Linux

Download Python from the official website:\
https://www.python.org/downloads/

Verify installation:

    python --version

or

    python3 --version

------------------------------------------------------------------------

## 💻 2. Install Visual Studio Code (VS Code)

Download VS Code (all platforms):\
https://code.visualstudio.com/

### Windows

During installation, enable: - Add to PATH\
- Open with Code (optional)

### macOS

Drag VS Code into the Applications folder.

### Linux

Depending on your distro:

#### Ubuntu / Debian

    sudo apt update
    sudo apt install code

#### Fedora

    sudo dnf install code

#### Arch Linux

    sudo pacman -S code

------------------------------------------------------------------------

## 🧩 3. Install Required Extensions

Open VS Code → Click **Extensions** → Install:

-   Python (Microsoft)\
-   Pylance\
-   Jupyter\
-   Code Runner (optional)

------------------------------------------------------------------------

## ⚙️ 4. Select Python Interpreter

1.  Press `Ctrl + Shift + P`\
2.  Type **Python: Select Interpreter**\
3.  Choose your Python installation or virtual environment

------------------------------------------------------------------------

## 🌐 5. Setting Up a Virtual Environment

### Windows

    python -m venv venv
    venv\Scripts\activate

### macOS / Linux

    python3 -m venv venv
    source venv/bin/activate

Select interpreter again in VS Code after activation.

------------------------------------------------------------------------

## ▶️ 6. Running Python Code

### Method 1 --- Run Button

Create `hello.py`:

``` python
print("Hello, VS Code!")
```

Click **Run ▶** at the top-right.

### Method 2 --- Terminal

Open terminal:

    Ctrl + `

Run:

    python hello.py

------------------------------------------------------------------------

## 🧪 7. Using Jupyter Notebooks (Optional)

Install **Jupyter Extension**\
Create a notebook via:

    File → New File → Jupyter Notebook

------------------------------------------------------------------------

## 🎨 8. Suggested Settings

Enable in Settings: - Format on Save\
- Auto Save\
- Linting\
- IntelliSense\
- Code Folding

(Optional)

    pip install black

------------------------------------------------------------------------


## 🎉 9. You're Ready!

VS Code is fully set up for Python across Windows, macOS, and Linux.

Start installing packages:

    pip install <package-name>
