## ğŸ 1. What Is Package Management?

Package management in Python refers to: - installing external
libraries, - updating packages, - removing unused packages, - isolating
project dependencies.

Python uses a rich ecosystem to manage packages efficiently.

------------------------------------------------------------------------

## ğŸ“¥ 2. Installing Packages with pip

`pip` is Python's default package manager.

### Check pip version:

    pip --version

### Install a package:

    pip install requests

### Upgrade a package:

    pip install --upgrade requests

### Uninstall a package:

    pip uninstall requests

### List installed packages:

    pip list

### Search packages:

    pip search <package-name>

## ğŸ§ª 4. Virtual Environments (venv)

A **virtual environment** is a self-contained environment that keeps
dependencies isolated.

### Create a virtual environment:

    python -m venv venv

### Activate:

#### Windows:

    venv\Scripts\activate

#### macOS / Linux:

    source venv/bin/activate

### Deactivate:

    deactivate

Using virtual environments prevents conflicts between projects.

------------------------------------------------------------------------

## ğŸš€ 5. pipx --- Installing CLI Tools

`pipx` is used to install Python-based CLI tools globally but in
isolated environments.

### Install pipx:

    pip install pipx
    pipx ensurepath

### Install a tool:

    pipx install black

### List installed pipx packages:

    pipx list

------------------------------------------------------------------------

## ğŸ“¦ 6. requirements.txt (Dependency File)

Commonly used to store project dependencies.

### Create a requirements file:

    pip freeze > requirements.txt

### Install dependencies from file:

    pip install -r requirements.txt

------------------------------------------------------------------------

## ğŸ§° 7. Poetry (Advanced Dependency Management)

**Poetry** is a modern dependency and package manager.

### Install Poetry:

    pip install poetry

### Initialize new project:

    poetry init

### Install dependencies:

    poetry add requests

Poetry automatically creates virtual environments and manages versions.

------------------------------------------------------------------------

## ğŸ“„ 8. Conda (Anaconda/Miniconda Users)

Conda manages both packages and environments, especially for data
science.

### Create environment:

    conda create -n myenv python=3.11

### Activate:

    conda activate myenv

### Install package:

    conda install numpy

------------------------------------------------------------------------

## ğŸ§¹ 9. Cleaning Up Packages

### Remove unused packages:

    pip uninstall <package>

### Remove and recreate environments when needed:

    rm -rf venv/
    python -m venv venv
